# User Guide

## What This Platform Does

Navigator helps users explore film and media theory through:

- searchable theory library
- category filters
- theory detail modal with key points, thinkers, films, and citations

## Main Areas

- Home
- Theory Library
- Vocabulary Hub
- Community/Admin placeholders

## How To Use

1. Open Theory Library.
2. Use search bar to find a topic.
3. Use category pills to narrow results.
4. Select Learn More on a theory card.
5. Review overview, context, key points, films, and citations.

## Notes

- Some theories have richer metadata than others.
- Modal action buttons are present and can be connected to future features.

## Feedback Guidance

When reporting issues, include:

- page URL
- theory title
- expected result
- actual result
- screenshot if possible
