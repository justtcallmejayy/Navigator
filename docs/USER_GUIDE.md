# Film Theory Navigator User Guide

Version: 1.1  
Last Updated: March 2026

## Quick Start

1. Open the app.
2. Go to Theory Library.
3. Search by keyword.
4. Open a theory.
5. Review concepts, films, and citations.

## Main Pages

- Home: overview and entry point.
- Theory Library: search and category filtering.
- Theory Detail: full theory content by slug.
- Vocabulary Hub: definitions and related theory links.
- Admin: restricted content management area.

## Common Workflows

### Find a theory quickly

1. Open Theory Library.
2. Enter a keyword.
3. Optional: apply one category filter.
4. Open Learn More.

### Build a bibliography

1. Open a theory detail page.
2. Go to citations.
3. Capture author, year, and title.
4. Complete full references in your citation manager.

## Troubleshooting

- No results: simplify keywords and clear filters.
- Blank page: refresh and retry in another browser.
- Layout issues: reset zoom to 100%.
- Missing content: data may still be unpublished.

## Admin Notes

- Publishing flow: Draft -> Review -> Published.
- Data publish command: npx tsx src/scripts/publish_data.ts
- Requires service-role key in environment.
